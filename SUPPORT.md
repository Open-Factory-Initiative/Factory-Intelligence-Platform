# Support

Factory Intelligence Platform is an early open-source project.

## Getting Help

Use GitHub Issues for:

- Bugs
- Documentation problems
- Feature requests
- Architecture questions
- Setup problems

Use GitHub Discussions if enabled for:

- Ideas
- Questions
- Use cases
- Roadmap discussion
- Contributor introductions

## Before Opening An Issue

Please include:

- What you were trying to do
- What happened
- What you expected
- Steps to reproduce
- Relevant logs or screenshots
- Your OS and environment
- Whether you are using simulator data or a real integration

Do not include secrets or proprietary plant data.
