# Prompt 03: Synthetic Factory Simulator

Act as a senior simulation engineer and manufacturing software engineer.

Goal: implement a deterministic synthetic factory simulator for local development and testing.

Simulator must support:

- One demo site
- One area
- One line
- Three to five assets
- Work orders
- Process measurements
- Quality measurements
- Seeded random generation
- Normal scenario
- Gradual drift scenario
- Sudden excursion scenario

Requirements:

- Use shared event contracts
- Provide CLI or script to generate scenario data
- Include unit tests
- Include fixture snapshots
- Document how to run scenarios
- Update `docs/LEARNING_LOG.md`

Do not implement ingestion service or UI in this prompt.
