# Prompt 13: Review and Refactor

Act as a senior software architecture reviewer.

Goal: review the repository for readiness after the first MVP implementation.

Read:

- AGENTS.md
- CODE_REVIEW.md
- docs/ARCHITECTURE.md
- docs/TESTING.md
- docs/GOVERNED_ACTIONS.md

Then inspect the implementation.

Report:

1. Architecture risks
2. Testing gaps
3. Documentation gaps
4. Safety/governance risks
5. Dependency concerns
6. Simplification opportunities
7. Suggested next PRs

Do not modify files until the review is complete and a plan is accepted.
