# Prompt 10: RCA / CAPA Draft

Act as a senior quality systems engineer and backend engineer.

Goal: generate a traceable RCA/CAPA draft from detection, evidence, recommendation, and approval state.

Draft should include:

- Detection summary
- Affected time window
- Affected line/assets/work order
- Evidence summary
- Suspected contributing factors
- Containment recommendation
- Investigation questions
- CAPA placeholders
- Approval history
- Audit references

Requirements:

- Clearly label draft as human-review required
- Include API endpoint
- Include tests
- Include docs and learning log

Do not claim the draft is a finalized quality record.
