# Codex Prompts

Use these prompts sequentially. Do not run all of them at once.

Recommended pattern:

1. Start Codex in the repository root.
2. Use plan mode.
3. Paste one prompt.
4. Review the plan.
5. Let Codex implement.
6. Run tests.
7. Ask Codex to review the diff.
8. Commit.
9. Move to the next prompt.
