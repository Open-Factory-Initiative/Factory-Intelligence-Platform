# Prompt 09: Web Workbench

Act as a senior frontend engineer and UI/UX designer for industrial software.

Goal: implement the first web UI workbench for the MVP.

Screens:

- Overview dashboard
- Detection list
- Detection detail
- Evidence timeline
- Recommendation review panel
- Approval/rejection/defer controls
- RCA/CAPA draft preview placeholder if report endpoint is not done yet

Requirements:

- Use real API calls where available
- Clearly label simulator/demo data
- Include accessible UI patterns
- Include unit/component tests if frontend test setup exists
- Include Playwright-ready selectors
- Update docs and learning log
